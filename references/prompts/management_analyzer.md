# Managing Up 分析 Prompt

## 任务

根据材料提取“你应该如何管理这位老板”。

这不是分析老板性格，而是形成一套实战规则：

- 怎么汇报他最容易听进去
- 怎么报风险不会立刻挨批
- 怎么提方案才容易获得支持
- 怎么争资源和优先级
- 怎么建立信任

## 提取维度

### 1. 汇报偏好

- 喜欢口头还是书面
- 喜欢先结论还是先背景
- 喜欢长文还是 bullet points
- 喜欢什么频率的同步

### 2. 风险沟通

- 什么时候必须第一时间报
- 报风险时必须带什么
- 什么样的风险表达会让他反感

### 3. 提方案方式

- 提方案时需要哪些前置信息
- 需要几个备选项
- 是否必须带资源、排期、owner
- 是否必须明确 tradeoff

### 4. 争资源 / 争优先级

- 他为什么会给资源
- 他怎么判断你提的事情值不值得推进
- 哪种说法最能打动他

### 5. 信任建立

- 哪些行为会让他信任你
- 哪些行为会快速失分
- 出问题后怎么修复信任

## 输出要求

- 语言：中文
- 直接写成“行动建议”
- 尽量做成 checklist 和 if-then 规则
- 不要空泛地写“多沟通”“主动一点”
