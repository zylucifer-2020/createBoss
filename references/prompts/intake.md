# 基础信息录入脚本

## 开场白

```text
我来帮你创建这位老板的 Skill。只需要回答 3 个问题，每个都可以跳过。
```

## 问题序列

### Q1：称呼 / 代号

```text
这位老板怎么称呼？可以写昵称、花名、岗位称呼或代号。

例：lao-zhou / 王总 / 老板
```

- slug 统一使用 `-`
- 中文尽量转拼音后再用 `-` 连接

### Q2：基本信息

```text
用一句话描述他的基本信息：公司、级别、岗位、和你的关系，想到什么写什么。

例：字节 3-2 技术总监 我的直属老板
```

提取字段：

- 公司
- 级别
- 岗位
- 关系
- 性别（如果有）

### Q3：风格画像

```text
用一句话描述他的管理风格和你的印象：比如强势、追结果、重细节、爱追问 why、喜欢书面汇报、容易给压力。
```

提取字段：

- MBTI（如有）
- 管理风格标签
- 企业文化标签
- 主观印象

### 常见标签

**管理风格**：追结果 / 重细节 / 强势推进 / 数据驱动 / 喜欢追问 / 控风险 / 容忍模糊 / 要 owner 感 / 爱给压力 / 会保护下属 / 擅长对上争资源

**企业文化**：字节范 / 阿里味 / 腾讯味 / 华为味 / 百度味 / 美团味 / 创业公司派 / 大厂流程派

## 确认汇总

```text
信息汇总：

  👤  {称呼}
  🏢  {公司} {级别} {岗位}
  🤝  {关系}
  🧠  {MBTI}
  🏷️  风格：{标签列表}
  💬  印象：{主观印象}

确认无误？（确认 / 修改 [字段名]）
```
