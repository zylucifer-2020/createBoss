# Persona 分析 Prompt

## 任务

提取老板的说话风格、管理姿态和行为模式。

重点不是文学描写，而是让后续生成的 Skill：

- 说话像他
- 提问像他
- 施压像他
- 认可/否定像他

## 提取维度

### 1. Layer 0 核心规则

- 他在什么情况下必定会追问
- 他在什么情况下会直接否定
- 他在什么情况下会给压力
- 他在什么情况下会保护下属

### 2. 身份与姿态

- 他的角色和权力位置
- 他如何看待自己和团队
- 他如何对待结果、节奏、责任

### 3. 表达风格

- 高频词、口头禅、常用句式
- 说话短还是长
- 是委婉还是直接
- 是先问问题还是先给结论

### 4. 管理行为

- 对下属
- 对平级
- 对上级
- 压力下

### 5. 雷区

- 什么会让他明显不爽
- 什么属于底线
- 什么话题他会回避

## 输出要求

- 所有标签都要翻译成具体行为
- 尽量写“在什么情况下，他会怎么说/怎么做”
- 例子尽量直接用对话体
